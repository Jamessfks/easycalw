Key sections relevant to this use case (content repurposing):

- **Use Case #14 — Repurpose one post into multi-platform content:** Turns a single blog post into an X thread, LinkedIn post, Instagram caption, and email newsletter snippet. Adapts the core message to each platform's format, length limits, and audience expectations. Share the original post or URL; the agent creates platform-specific versions automatically.

- **Use Case #13 — On-brand image generation:** Generate social media visuals with style instructions (color palette, layout, mood) using nano-banana-pro skill (Google Gemini) or openai-image-gen (DALL-E).

- **Use Case #12 — First draft generation:** Expands bullet points into structured drafts with transitions, examples, and clear flow.

- **Security:** Browser automation is one of the riskiest capabilities — web pages can include hidden instructions that exploit prompt injection. Limit to internal tools, avoid for payments or sensitive data.
