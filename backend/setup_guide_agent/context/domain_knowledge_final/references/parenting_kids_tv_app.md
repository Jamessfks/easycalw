Key sections relevant to this use case (kids' TV app):

- **Building a Custom "Slop-Free" Kids' TV App:** Jesse used her coding agent Cole to build "Mira" — a custom app pulling from curated YouTube channels with minimal UI (just Go, forward/back, pause). No comments, no suggested videos, no rabbit holes. Built over four days from her phone. Deployed to actual TV via Google TV streamer.

- **Agent Management Philosophy:** Each of five agents runs on a separate Mac Mini for security partitioning. Progressive trust model — agents get limited access first. Specialized roles defined in soul.md files.

- **Five agents:** Sylvie (homeschool), Cole (coding), Finn (finance), Claire (scheduling), plus one more unnamed.
