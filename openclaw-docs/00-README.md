## How to use

When working with OpenClaw configuration or setup:
1. Start with **02-getting-started.md** for initial setup
2. Use **04-configuration.md** for common configuration tasks
3. Refer to **05-configuration-reference.md** for specific field details
4. See **06-configuration-examples.md** for copy-paste ready configs
5. Check **10-cli-reference.md** for CLI commands and options

For the Google ADK codelab integration this project builds on, see files 12 and 13.
