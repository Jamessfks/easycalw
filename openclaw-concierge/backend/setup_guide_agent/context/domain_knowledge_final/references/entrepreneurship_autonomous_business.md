## Top Takeaway: 3-Layer Memory System
1. **Layer 1: Knowledge graph** — ~/life/ folder using PARA system (projects, areas, resources, archives). Durable facts about people and projects with summary files.
2. **Layer 2: Daily notes** — Dated markdown file for each day. Bot writes during conversations, extracts important stuff into Layer 1 during nightly consolidation.
3. **Layer 3: Tacit knowledge** — Facts about you: communication preferences, workflow habits, hard rules, lessons learned.
